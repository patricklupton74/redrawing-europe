# redrawing-europe 🗺️

WIP — Training ML algorithms from scratch to rediscover European national borders using only lat/lng data.

## Progress
- World city data filtered and cleaned to 38 European countries
- kNN implemented from scratch 
- K-Means implemented from scratch
